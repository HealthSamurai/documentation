# Aidbox version

Aidbox version is available to obtain with the following `REST` method:&#x20;

{% tabs %}
{% tab title="Request" %}
```http
GET /$version
```
{% endtab %}

```
commit: c51312c6
zen-fhir-version: 0.5.8
```
{% endcode %}
